# admin-template

## 安装依赖
```
npm install
or
npm install --registry https://registry.npm.taobao.org
```

### 本地运行
```
npm run serve
```

### 打包-生产
```
npm run build
```

### 打包-开发
```
npm run build:dev
```

### 打包-性能分析
```
npm run build:report
```

### Lints and fixes files
